Mixcloud (with) Tracklist is back!

Now, you can access and interact with tracklists through the extension's browser popup.
(Due to changes on Mixcloud website, tracklists can no longer be directly integrated into pages).

Simply click the extension icon to view the tracklist. You can still conveniently play tracks directly from this popup.
The displayed tracklist is from your first open Mixcloud tab (if you have multiple tabs open).
Tracklists from multiple tabs will be managed in a future update.
For more infos, check out the Chrome web store page.

One last point, Chrome by default hide extensions in its extension menu (this icon )
