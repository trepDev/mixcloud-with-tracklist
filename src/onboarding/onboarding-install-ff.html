Mixcloud (with) Tracklist

You can access and interact with tracklists through the extension's browser popup.

Simply click the extension icon to view the tracklist. You can conveniently play tracks directly from this popup (If the DJ did provide a timestamp).
The displayed tracklist is from your first open Mixcloud tab (if you have multiple tabs open).
Tracklists from multiple tabs will be managed in a future update.

For more infos, check out the Firefox addon page.
